## 🎓 Key Learnings & Patterns

### Mobile Responsive Pattern Used
```tsx
// Mobile menu button (hidden on desktop)
<div className="md:hidden"> {/* Hamburger icon */} </div>

// Mobile overlay
{isMobileSidebarOpen && <div className="md:hidden"> {/* overlay */} </div>}

// Desktop sidebar (hidden on mobile)
<div className="hidden md:flex"> {/* Sidebar */} </div>

// Mobile sidebar sheet (hidden on desktop)
<div className={`md:hidden ${isMobileSidebarOpen ? 'translate-x-0' : 'translate-x-full'}`}>

// Responsive main content
<div className="mr-0 md:mr-16"> {/* Content */} </div>
```

### Responsive Breakpoints
- `md: 768px` - Main breakpoint for mobile to desktop
- `md:hidden` - Hide on desktop, show on mobile
- `hidden md:flex` - Show on desktop, hide on mobile
- `mr-0 md:mr-16` - 0 margin on mobile, 4rem on desktop

### RTL Implementation
- Add `dir="rtl"` to root container
- Use `space-x-reverse` on flex containers
- Sidebar naturally slides from right with CSS transforms

### Dark Mode Support
- Use `dark:` prefix for dark colors
- Applied to all components
- Smooth transitions when theme changes
