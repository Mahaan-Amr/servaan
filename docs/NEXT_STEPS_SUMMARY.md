#### **Frontend نیاز دارد:**
```typescript
// مسیرهای CRM که باید پیاده‌سازی شوند:
/workspaces/customer-relationship-management/
├── customers/          // مدیریت مشتریان
│   ├── page.tsx       // لیست مشتریان

├── campaigns/          // مدیریت کمپین‌ها
├── analytics/          // تحلیل CRM
└── visits/            // ردیابی بازدیدها
```
